📖 사용 방법

① 원 배치

  • 아래 트레이에서 번호 원을 필드 위로 드래그
  • 필드 안에서 드래그 → 위치 이동
  • 필드 밖으로 드래그 → 트레이로 복귀
  • 🔺 역V 대형 — 24명 자동 배치
  • 🗑 비우기 — 전원 트레이로

② 동선 저장

  • 배치 완료 후 ⊕ 지금 위치 저장 클릭 → KF1 생성
  • 원들을 다음 위치로 이동 → 다시 ⊕ 지금 위치 저장 → KF2 생성
  • 반복해서 동선을 완성 (최소 2개 필요)
  • 아래 KF 칩 누르면 그 배치로 되돌아감
  • KF 삭제 — 저장된 동선 전부 삭제

③ 동선 재생

  • ▶ 동선 재생 — KF 순서대로 원들이 부드럽게 이동
  • 이동 궤적이 네온 선으로 표시됨
  • ■ 중지 — 재생 종료

원 색상

  • ⚪ 1·2번 — 리더 (역V 맨 앞)
  • 🟣 홀수번 — 왼쪽 날개
  • 🔵 짝수번 — 오른쪽 날개

📱 핸드폰에서 사용하기

  • 이 파일(choreography.html)을 카카오톡이나 AirDrop으로 핸드폰에 전송
  • 핸드폰에서 파일을 열 때 Safari(아이폰) 또는 Chrome(안드로이드)으로 열기
  • 손가락으로 원을 드래그해서 배치 가능
  • 가로 모드로 돌리면 필드가 더 넓게 보여요
